Key Attributes : ekta entity or table er jei attribute unique data hold kore seta Key Attribute . jemon ekta table er name key attribute hote pare na but phone number, email egula key attribute hote pare 

key attribute abar 3 prokar 
1. Primary Key : jei attribute ke oi table er unique attribute or primary attribute banano hoy jate ekta data uniquely identify kora jay seta Primary Key 
2. Candidate Key : ekta table e onk gula unique data wala attribute thakte pare , jemon phone number , email , studentId, Nid number etc. but ami to only 1 ta attribute ke primary key banate parbo tahole baki attribute gula tokhon Candidate key hishebe thakbe .
